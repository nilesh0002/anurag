# E-Commerce Web Application (Secure Auth + Admin Panel)

## Project Overview

This project is a modernized version of a traditional HTML-based e-commerce website developed as part of an academic practical file. The original project used outdated technologies like frames, tables, and marquee tags.

This version transforms the project into a responsive, mobile-friendly, and secure web application with a clean UI inspired by modern websites like McDonald's.

---

## Key Features

### Secure Authentication System

- User Registration (Email/ID + Password)
- OTP-based Login using Authenticator App
- Multi-step verification for enhanced security

### Admin Panel

- View all registered users
- Delete users securely
- Prevent deletion of admin account (protected role)
- Reuse email after user deletion

### Mobile Responsive UI

- Fully optimized for mobile devices
- Clean layout using Flexbox/Grid
- No overlapping or UI issues
- Touch-friendly buttons and navigation

### Modern UI/UX

- Inspired by real-world websites
- Smooth scrolling and hover effects
- Card-based product layout
- Clean typography and spacing

### Website Sections

- Home (Hero Section)
- Products / Gallery
- About Section (Content Page)
- Video Section
- Contact Section
- Map Integration (Google Maps)
- User Form (Input handling)
